# 개발 연습 폴더 안내

이 폴더는 **Next.js + Supabase + Vercel** 조합으로 웹 서비스를 만들어보는 연습 공간입니다.

---

## 이 폴더에 있는 것들

| 폴더/파일 | 설명 |
|-----------|------|
| `index.html` | 가계부 앱 (단일 HTML 파일) |
| `goguma-market/` | 고구마마켓 앱 (Next.js 프로젝트) |

---

## 사용 중인 도구 설명

- **Next.js** — 웹페이지를 만드는 틀(프레임워크). 리액트 기반으로, 파일을 만들면 자동으로 페이지가 생깁니다.
- **Supabase** — 데이터를 저장하는 온라인 창고(데이터베이스). 회원 관리, 데이터 저장 등을 담당합니다.
- **Vercel** — 만든 웹사이트를 인터넷에 올려주는 서비스(배포 플랫폼). 코드를 올리면 자동으로 웹사이트가 됩니다.

---

## 작업 원칙

1. **사용자가 지시한 것만** 단계적으로 수행합니다.
2. 자동으로 할 수 있는 것은 최대한 처리합니다.
3. **꼭 사람이 직접 해야 하는 것**(비밀번호 입력, 클릭 등)만 쉬운 말로 안내합니다.
4. 작업 중 무엇을 하고 있는지 자주 알려줍니다.
5. 개발 용어는 비개발자도 이해할 수 있게 설명합니다.
6. 개발이 끝나면 **바로 결과를 확인**할 수 있게 해줍니다.
